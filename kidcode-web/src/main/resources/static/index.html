KidCode Language Reference

                
Welcome to KidCode! This guide will teach you everything you need to know to control Cody and create amazing drawings.

                
1. Basic Turtle Commands

                
CommandSyntaxExampleDescription
Movemove forward <value>move forward 100Moves Cody forward by a number of pixels.
Turnturn <direction> <value>turn right 90Turns Cody to the left or right by a number of degrees.
Saysay <value>say "Hello!"Displays a message in the output log. Can be a string or a variable.

                
2. Pen and Color Control

                
CommandSyntaxExampleDescription
Penpen <up / down>pen upLifts the pen (to move without drawing) or lowers it (to draw).
Colorcolor <color_name>color "red"Changes the color of the pen.

                
Supported Colors: "red", "green", "blue", "yellow", "orange", "purple", "black", "white".

                
3. Variables and Expressions

                
Variables
Use the set command to create a variable and give it a value.
  • Syntax: set <variable_name> = <value>
  • Example: set size = 50

4. Control Flow

                
Loops (repeat)
repeat <times>
    # code to repeat
end repeat

                
Conditionals (if/else)
if <condition>
    # code
else
    # other code
end if

                
5. Functions
Group your code into reusable commands.
define square size
    repeat 4
        move forward size
        turn right 90
    end repeat
end define

square 50

                
6. Lists
Store a collection of items. Access with list_name[0].
set colors = ["red", "green", "blue"]
color colors[0]
